# Clinical Lab Test Tracker

Clinical Lab Test Tracker is a React-based single-page application designed to simulate a clinical laboratory workflow. The application allows users to log, manage, and prioritize patient laboratory tests through an intuitive dashboard interface.

## Features

- **Automated Accessioning:** Automatically generates sequential Lab IDs for incoming samples (example: LAB-001).
- **Dynamic Test Queue:** Features a real-time updating data table for tracking pending patient tests.
- **Priority Tagging:** Uses dynamic visual badges to distinguish between Urgent and Routine test requests.
- **Workflow Overview:** Dashboard statistics display the total pending tests and urgent sample counts.
- **Modular Design:** Constructed using 11 highly reusable custom React components (including custom Inputs, Selects, Forms, DataTables, and Alerts) for maximum scalability.

## Technologies Used

- **React.js:** Functional components, Props, and State Management (`useState`)
- **Vite:** Next-generation frontend tooling
- **CSS:** Inline React styling combined with standard CSS (Flexbox, CSS Grid, and custom gradients)

## Local Setup

To run this project on your local machine, follow these steps:

1. Clone the repository to your local machine.
2. Open the directory in your terminal.
3. Install the required dependencies:

   ```bash
   npm install
   npm run dev
   ```
